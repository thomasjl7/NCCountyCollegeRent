# This Bivariate map showcases the location and type of each college in North Caroline, while also displaying the median rent of each county in North Carolin from 2023. The main goal of this mpa is to help infer if colleges can influence the rent in a county, and if so, which type of colleges have a greater or lesser impact on rent. The code for this map primarily uses HTML, Javascript, and CSS. The data sourced for this map comes from NC OneMap and the United States Census Bureau, with the Bureau providng the county shapefile and rent numbers, while NC OneMap provided the college data. Each college should have a pop up feature showcasing the name of the college, and the legend should help give a range of each counties median rent.
